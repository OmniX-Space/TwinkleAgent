# TwinkleAgent ⭐
Built by MoeCinamo, this AI agent is capable of self-improvement and task execution. Developed with C++, this means you can run on any embedded device with minimal performance loss. It has a built-in learning loop and can also create skills from experience, continuously improve during use, promote continuous mastery of knowledge, search for past conversations, and build deeper self models in each conversation. It may not necessarily be on one computer, it could also be a cluster, and you can communicate with it through chat apps such as WeChat/Telegram.

Use any model you want. As long as it is compatible with the OpenAI protocol.
